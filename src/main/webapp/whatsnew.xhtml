What's new?

Version 1.0.10

Web Sheet

  • Fix: attr Rendered problem.[issues17]. The rendered attribute in component tag doesn't work as expected:
    • Removed the rendered attribute from component tag.
    • Added the rendered attribute into webSheetBean. You can use setRendered(false) in the bean to hide the web sheet. Please see the "Validation" demo page for detail.

Version 1.0.9

Web Sheet

  • Fix: translate labels in modal of remove line.[issues18]. The following attributes have been added into component tag:
    • deleteRowConfirmHeader: String to set the header title of confirmation dialog for delete row. default is "Confirmation". required="false"
    • deleteRowConfirmButtonYes: String to set the title of "Yes" button in delete row confirmation dialog. default is "Yes". required="false"
    • deleteRowConfirmButtonNo: String to set the title of "No" button in delete row confirmation dialog. default is "No". required="false"
    • addRowButtonTip: String to set the tool tip of add row button. default is "Add Row". required="false"
    • deleteRowButtonTip: String to set the tool tip of delete row button. default is "Delete Row". required="false"

Version 1.0.0

Web Sheet

  • Tiefaces now only focus on web sheet development. Other components have been removed.
  • Uploaded the tiefaces artifact into Maven central repository.
  • Changed the way to define template.
  • Use JEXL to evaluate data.
  • Use special markup in the cell comment to support special features.
  • Support nested collection data.
  • Two way data binding between data object and web sheet cell.
  • Support multiple Primeface input widget for cell input method (i.e. calandar, dropdown, inputNumber, inputArea..).
