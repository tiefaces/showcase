Web Sheet Tag

Tiefaces use tie:websheet tag in the JSF web page to define the component.

How to define?

Use the special syntax below defined in the comment for the staring cell of the command.

...
<tie:websheet id="sheet1"
....
>
</tie:websheet >
...

The Attributes of tie:websheet

  • webSheetBean: The bean object which passed in. required="true"
  • readOnly: True then not editable. required="false"
  • showLineNumber: True then show line number. required="false"
  • paginator: Same as p:datatable. required="false"
  • paginatorTemplate: Same as p:datatable. required="false"
  • paginatorPosition: Same as p:datatable. required="false"
  • paginatorAlwaysVisible: Same as p:datatable. required="false"
  • maxRowsPerPage: Same as p:datatable. required="false"
  • rowsPerPageTemplate: Same as p:datatable. required="false"
  • tableWidthStyle: Same as p:datatable. required="false"
  • emptyMessage: Same as p:datatable.
  • deleteRowConfirmMessage: confirmation message when prompt delete row. required="false"
  • hideSingleSheetTabTitle: default is false. When set to true, If there's only one form, then hide the form title. required="false"
  • deleteRowConfirmHeader: String to set the header title of confirmation dialog for delete row. default is "Confirmation". required="false"
  • deleteRowConfirmButtonYes: String to set the title of "Yes" button in delete row confirmation dialog. default is "Yes". required="false"
  • deleteRowConfirmButtonNo: String to set the title of "No" button in delete row confirmation dialog. default is "No". required="false"
  • addRowButtonTip: String to set the tool tip of add row button. default is "Add Row". required="false"
  • deleteRowButtonTip: String to set the tool tip of delete row button. default is "Delete Row". required="false"
