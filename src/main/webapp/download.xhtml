How to use it in maven project?

The TieFaces haven't been uploaded into maven center repository, so now you only could use it as your local library.

For example:

1. Save the jar file into your project:

2. pom.xml:

<repositories>
    <repository>
        <id>ProjectRepo</id>
        <name>ProjectRepo</name>
        <url>file://${project.basedir}/src/lib</url>
    </repository>
</repositories>

<dependencies>
	<dependency>
         <groupId>tiefaces</groupId>
         <artifactId>tiefaces</artifactId>
         <version>0.2.4</version>
      </dependency>		
</dependencies>
