Download

The easiest way to add Tiefaces libraries to your project is to use Maven and specify the required libraries in pom.xml. 

<dependency>
	    <groupId>org.tiefaces</groupId>
	    <artifactId>tiefaces</artifactId>
	    <version>1.0.0</version>
</dependency>

Alternatively you can download Tiefaces distribution from the link below:

  • Tiefaces #{_tieFacesVersion} library
  • Tiefaces #{_tieFacesVersion} source
  • Tiefaces #{_tieFacesVersion} javadoc

Also you need manage the following dependences manually:

  • Primefaces 6.0 or up
  • Poi 3.15 or up (i.e. poi-3.15.jar poi-ooxml-3.15.jar poi-ooxml-schemas-3.15.jar xmlbeans-2.6.0.jar commons-collections4-4.1.jar)
  • Jfreechart 1.0.19 (i.e. jfreechart-1.0.19.jar jcommon-1.0.23.jar)
  • Jexl 2.11 (i.e. commons-jexl-2.1.1.jar)
