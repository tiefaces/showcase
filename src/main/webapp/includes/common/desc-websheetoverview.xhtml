What is TieFaces WebSheet

Tiefaces webSheet is a JSF component which can render a excel template as a web form while keeping same formats and styles. It is not intend to become online document tools like Google Docs, instead it want to become a gateway or transformer of Excel sheets to Web forms.

Features

Render excel templates as web forms

  • Support multiple sheets
  • Keep Excel formats and styles
  • Keep Excel formulas
  • Support charts and images

Dynamic Web Form Solution

  • Input/Output data through context object.
  • Support multiple input/output widget. i.e. calendar, dropdown, inputNumber, inputArea etc.
  • Support recursive data structure. i.e. each command.
